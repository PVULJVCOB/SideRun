Siderun

A lightweight flying-border runner

A small JS + CSS effect that draws an animated border and blurred backdrop. Attach it to navs, logos and cards; it follows hover or the pointer for a smooth running highlight.

siderun

Hover inside the box to see the siderun move based on link/host hover. This demo uses the stroke host markup and initializes the front-end effect.

// HTML host needs a stroke container
<div id="siderun" class="sr-container glass-bg">
  <div class="site-nav__stroke siderun"></div>
  ...
</div>

// JS init (siderun: hover-driven, like the logo)
SideRun.init(document.getElementById('siderun'), {
  radius: 12,
  tail: 14,
  margin: 11
});

siderun-follow

Move the cursor — the siderun-follow tracks the pointer smoothly. Toggle trackPointer to enable direct pointer tracking.

// HTML host needs a stroke container
<div id="siderun-follow" class="sr-container glass-bg">
  <div class="site-nav__stroke siderun"></div>
  ...
</div>

// JS init (siderun-follow: cursor tracking)
SideRun.init(document.getElementById('siderun-follow'), {
  radius: 12,
  tail: 18,
  margin: 11,
  trackPointer: true,
  ease: 0.08
});
SIDERUN

A lightweight front-end runner effect

A tiny JS runtime plus companion CSS that provides a running border and blurred backdrop. Designed for predictable visuals and easy theming.

// Front-end quickstart
// Easiest: copy files from `SideRun/` (siderun.js, siderun.css)
// or copy these two files directly from the `SideRun/` folder in this repo.

// HTML host needs a stroke container
<div id="siderun" class="sr-container glass-bg">
  <div class="site-nav__stroke siderun"></div>
  <!-- content -->
</div>

// JS init (siderun)
SideRun.init(document.getElementById('siderun'), {
  radius: 12,
  tail: 14,
  margin: 11
});

How it works

  • Runtime injects an SVG frame and a blurred backdrop into a stroke host.
  • Positioning can follow anchors (link hover) or direct pointer tracking.
  • Animation uses stroke-dasharray/offset and per-frame interpolation; respects prefers-reduced-motion.

Accessibility: the effect is decorative — keep focus outlines for keyboard users and respect reduced-motion preferences.

Purpose

Provide a visually appealing, low-cost animated border/runner for UI elements. Ideal for navs, logos, and content cards.

Install

Front-end integration: copy SideRun/siderun.js and SideRun/siderun.css into your project, then include and initialize. No server components required.
