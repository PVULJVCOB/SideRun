SideRun

Hover to see the animated border effect that follows your mouse. The border smoothly animates around the element with customizable speed, color, and radius.

// HTML host needs a stroke container
<div id="siderun" class="sr-container glass-bg">
  <div class="site-nav__stroke siderun"></div>
  ...
</div>

// JS init (siderun: hover-driven, like the logo)
SideRun.init(document.getElementById('siderun'), {
  radius: 12,
  tail: 14,
  margin: 11
});

Pointer Tracking

Move the cursor — the siderun-follow tracks the pointer smoothly. Toggle trackPointer to enable direct pointer tracking.

// HTML host needs a stroke container
<div id="siderun-follow" class="sr-container glass-bg">
  <div class="site-nav__stroke siderun"></div>
  ...
</div>

// JS init (siderun-follow: cursor tracking)
SideRun.init(document.getElementById('siderun-follow'), {
  radius: 12,
  tail: 18,
  margin: 11,
  trackPointer: true,
  ease: 0.08
});

Modern Architecture

  • CSS Token System: Full control through custom properties with responsive scaling.
  • 3-Layer Design: Separates blur effects, border animations, and interactive content.
  • Touch Optimized: Automatic device detection with proper touch feedback and hover cleanup.
  • Responsive Design: Seamless desktop-to-mobile transitions with safe-area support.

Accessibility: Full keyboard navigation, screen reader support, and respects reduced-motion preferences. GPU-accelerated for smooth performance.

Features

Lightweight flying border - responsive design, touch optimization, CSS tokens, accessibility features, and GPU-accelerated animations. Perfect for interactive elements.

Quick Start

Copy siderun.js and siderun.css. Add a host container (e.g., div.sr-container) with a child .site-nav__stroke.siderun, then initialize the host with SideRun.init(host). Customize via options.
